### Usage

#### Compile 1

`javac Counter.java Flips.java FlipsMax.java Rolls.java`

#### Counter

`java Counter`

#### Flips

`java Flips 10000`

#### FlipsMax

`java FlipsMax 10000`

#### Rolls

`java Rolls 100000`

#### Compile 2

`javac Cat.java Date.java`

#### Cat

`java Cat In1.txt In2.txt Out.txt`

#### Date

`java Date 7 27 1991`

`java Date 7/27/1991`
